Nintendo Entertainment System (1985)

Welcome to the NES game library. On this site, you will find a comprehensive list of liscensed games produced for the NES from 1985 to 1995.

More on NES game production:

NES games were stored on cartridges, which contained ROM chips that held the game data. These cartridges had a set of metal pins that connected to the console's cartridge slot, allowing the NES to read the data and load the game. This pin-based connection was crucial for the system's functionality, enabling quick, reliable game loading and easy swapping of games.
